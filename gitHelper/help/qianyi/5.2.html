Git客户端迁移（参考)
从git获取代码：
git clone 仓库地址

获取全部分支 
git branch -r | grep -v '\->' | while read remote; do git branch --track “$remote#origin/}” “$remote”; done
git fetch –all
git pull –all

提交代码到代码托管模块
添加远端仓库地址并推送：

git push -u origin master(远端指定分支)
若需要推送所有分支，使用
git push REMOTE '*:*'
git push REMOTE --all
git push --all origin
提交代码到代码托管模块
添加远端仓库地址并推送：
git remote set-url origin 仓库地址



git push -u origin master



仓库的关联配置库变更

点击左侧菜单【代码工程】,选择需要变更的仓库，点击【管理】下菜单【基础设置】。选择关联配置库。点击【保存】按钮进行保存。
