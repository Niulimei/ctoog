掌握代Git基本命令操作

代码托管平台Git操作命令，第一次使用需要:初始化本地仓库、配置SSH公钥、克隆仓库、配置仓库远端url等,日常开发中主要使用的是:推、拉、提、改等。


Gitee操作命令，第一次使用需要:初始化本地仓库、配置SSH公钥、克隆仓库、配置仓库远端url等,日常开发中主要使用的是:推、拉、提、改等。详细命令见下文：

初始化本地git仓库:
git init

配置SSH公钥：
ssh-keygen 
或者
ssh-keygen -t rsa -C “自己的邮箱地址”

克隆仓库：
git clone http://*****.git;

配置仓库远端URL:
git remote add origin 代码托管模块仓库克隆地址

推送到远端：
git push -u origin master

拉取代码(当前分支下）
git pull 

切换分支:
git checkout -b 分支名

添加修改文件到暂存区：
git add 文件名

提交代码：
git commit -m “提交信息”

推送修改内容到远端：
git push origin 分支名
